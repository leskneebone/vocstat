# Contributing to VocStat

VocStat is a personal project by Les Kneebone. Thoughtful feedback is welcome,
especially where it helps clarify the specification, identify relevant sources,
or improve the usefulness of vocabulary statistics.

## Issues

Issues are the preferred way to contribute at this stage.

Good issues might include:

- a source, standard, paper, or report that VocStat should consider;
- a proposed statistic or visualisation;
- a possible SHACL observation;
- a bug or ambiguity in the current code, shapes, or documentation;
- a design question about how vocabulary information should be reported.

Please include links, citations, examples, or small RDF snippets where they help.

## Pull Requests

Pull requests are welcome by prior discussion. Please open an issue first unless
the change is very small, such as fixing a typo.

By submitting a pull request, you confirm that you have the right to contribute
the material and that you grant permission for it to be included in VocStat.
Submitting a pull request does not transfer project ownership, governance, or
maintainership.

## Direction

VocStat is not currently a community-led project. Feedback and contributions may
inform the work, but project direction and final editorial decisions remain with
Les Kneebone.
